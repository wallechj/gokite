# gokite

gokite 定时运行



下载代码：
```bash
git clone https://github.com/wallechj/gokite.git
```
```bash
cd gokite
```

安装依赖：
```bash
pip install -r requirements.txt
```

打开walletAddress.txt填写钱包地址

```bash
0x........
```
proxies.txt（如果需要，可不填）

```bash
http://1.2.3.4:8080:user:pass
socks5://5.6.7.8:1080
1.2.3.4:8080:user:pass
```


运行代码：
```bash
python bot.py
```
或
```bash
python3 bot.py
```
