# gokite

gokite 定时运行



下载代码：
```bash
git clone https://github.com/wallechj/gokite.git
```
```bash
cd gokite
```

安装依赖：
```bash
pip install -r requirements.txt
```

打开wallets.txt填写钱包地址

```bash
0x........
```
proxies.txt（如果需要，可不填）

```bash
socks5://5.6.7.8:1080
http://user:pass@9.10.11.12:8080
```


运行代码：
```bash
python bot.py
```
或
```bash
python3 bot.py
```
